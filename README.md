# 🏢 MRD ENTERPRISES | Official Citizen Service Hub

![Thane Janseva Portal](https://images.unsplash.com/photo-1450101499163-c8848c66ca85?w=1200&auto=format&fit=crop&q=80)

> A high-fidelity, production-grade, and multi-language business portal specifically engineered for **Thane's Official Documentation & Janseva Services**.

---

## 🎯 Project Vision
**MRD Enterprises** is a modern documentation service hub designed to provide citizens with a seamless, high-contrast, and mobile-first experience for official government registrations and certificates. The portal is built with a focus on absolute clarity, multi-language transparency, and extreme visual stability.

## ✨ Key Features
- **🌐 Triple Language Hub**: 100% dynamic support for **English**, **Hindi (हिन्दी)**, and **Marathi (मराठी)** across all UI segments.
- **📱 Extreme Responsive Design**: Pixel-perfect layout for all devices with a dedicated tactical mobile bottom action bar.
- **⚖️ Tactical Design System**: Strict adherence to a **Pure Black (#000000)**, **Orange (#f97316)**, and **White (#ffffff)** color palette.
- **⚡ Performance First**: Powered by Next.js 14 for lightning-fast server-side rendering and static optimization.
- **🗺️ Official Integration**: Custom Google Maps embedding for the Kolshet Road, Thane service hub.

## 🛠️ Technology Stack
- **Framework**: [Next.js 14](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Typography**: 
  - **Headings**: Poppins (Bold & Tactical)
  - **Body/UI**: Inter (Clean & Legible)
- **Localization**: Custom dynamic JSON-based message hub.

## 🚀 Getting Started

### 1. Installation
Install dependencies using npm:
```bash
npm install
```

### 2. Development Mode
Run the development server locally:
```bash
npm run dev
```
Open `http://localhost:3000` to preview.

### 3. Production Build
Generate an optimized production bundle:
```bash
npm run build
```

## 🎨 Design System Standards
The application follows a rigorous visual hierarchy defined in `globals.css`:
- **Typography**: Headings use `font-black` with uppercase tracking. Paragraphs use bold weights for maximum legibility.
- **Colors**: Strictly **Black/Orange/White**. Gray tones have been eliminated to ensure a premium government-portal aesthetic.
- **Borders**: Tactical `border-2` or `border-4` black borders used to establish clear visual containers.
